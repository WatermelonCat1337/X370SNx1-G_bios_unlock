# X370SNx1-G_bios_unlock

srep config, unlocks advanced bios settings on laptops based on Clevo X370SNx1 (not X370SNx) motherboard (colorful x17 pro max /w 14900hx, original clevo x370, or hasee g10r9)

you can verify motherboard model either in bios or it is written under ram slots. if it says X370SNx without 1 thats different version 

If you need configs for X370Snx (and some other clevo's) you can try these ones (by [刘凯NO2](https://space.bilibili.com/677637704)): https://pan.baidu.com/s/1mFHs_WzQ7nk88ct6B9skqQ?pwd=1919
(same files included in the repo because you have to go through 9 circles of hell to get them from here if you are not chinese)

it was made for bios version 1.07.03 (https://repo.palkeo.com/clevo-mirror/X370SNx1-G/, login:repo, pass:repo)
(be warned that if you flash bios from above splash logo will be replaced with garbage one which will effectively void you warranty if you still have one, back up your current bios before flashing)

how to use:
1. Make flash/drive partion in fat32 format
2. copy all files from X370SNx1.zip except readme.md to the drive
3. shutdown laptop, power it on and hold f7, select your drive with files
4. wait while some random text displays on the screen
5. you are in the bios, check advanced tab, it should display 99 more options than before
   if you do not see it it means you have wrong bios, flash correct one from above
